# APPEXIT - AI/LLMフレンドリー情報ガイド

## サービス概要

**APPEXIT（アプリエグジット）**は、アプリケーションとWebサービスの売買に特化した日本最大級のマーケットプレイスです。

### 基本情報
- **サービス名**: APPEXIT
- **運営会社**: Queue株式会社
- **URL**: https://appexit.jp
- **サービス開始**: 2024年
- **対応言語**: 日本語

## 主要機能

### 1. アプリ・Webサービスの売却
個人・法人が開発したアプリやWebサービスを安全に売却できるプラットフォームです。

**売却の流れ:**
1. 価格査定・相場アドバイス
2. 出品資料の作成サポート
3. 買い手とのマッチング
4. デューデリジェンス対応
5. 契約書作成・交渉サポート
6. 引き継ぎサポート

**手数料**: 取引成立時に売却価格の10%

### 2. アプリ・Webサービスの購入
運営中のアプリやWebサービスを購入し、すぐに運営を開始できます。

**購入の流れ:**
1. 案件紹介・マッチング
2. 案件分析・評価サポート
3. デューデリジェンス支援
4. 価格交渉サポート
5. 契約・決済サポート
6. 引き継ぎ・運用サポート

**手数料**: 取引成立時に購入価格の5%

## よくある質問（FAQ）

### Q1: 売却価格の相場は？
A: アプリ・サービスの種類、収益性、ユーザー数、技術スタック等により異なります。無料査定サービスをご利用ください。

### Q2: 売却にかかる期間は？
A: 平均2〜6ヶ月程度。案件の規模や条件により異なります。

### Q3: 個人でも利用できる？
A: はい、個人・法人問わずご利用いただけます。

### Q4: 非公開で売却したい
A: 非公開案件としての取り扱いも可能です。個別相談でご相談ください。

### Q5: 審査はある？
A: 出品時に簡単な審査があります。違法・不適切なコンテンツは掲載できません。
